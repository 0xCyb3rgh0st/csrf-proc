# Privacy

CSRForge is designed as a local-first static application.

- Requests are processed locally in the browser.
- Projects are stored in browser storage through IndexedDB.
- No account is required.
- No request data is uploaded by the core application.
- Clearing site data removes locally stored projects.
- Exported project files are controlled by the user.
- Sensitive data may be included in saved or exported data only when explicitly selected.

The core application does not include third-party analytics or error reporting services.
