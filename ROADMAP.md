# Roadmap

1. Foundation: static app, parser, inspector, local storage, project import/export.
2. Analysis: browser feasibility, cookies, SameSite, CSRF defenses, explainable findings.
3. Generation: browser-realistic PoC templates, safe preview, controlled test flow.
4. Reports: Markdown, HTML, JSON, and ZIP evidence exports.
5. Quality: workers, PWA, accessibility, performance, fuzzing, E2E tests.
